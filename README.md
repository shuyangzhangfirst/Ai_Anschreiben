Dieses Projekt dient dazu, Bewerbern dabei zu helfen, schnell ein Anschreiben (Anschreiben) zu generieren.
Das Programm wird durch die Ausführung der Datei aianschreiben.py gestartet.
Außerdem wird ein Gemini-API-Key benötigt, um das Programm auszuführen.

Derzeit befindet sich das Projekt noch in der Entwicklungs- und Verbesserungsphase.
